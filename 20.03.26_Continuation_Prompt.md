## Known Issue: Next.js 16 Middleware Deprecation

`frontend/src/middleware.js` uses `export { default } from 'next-auth/middleware'` which Next.js 16 deprecated. The build compiles but fails at page data collection with:

```
Error: The file "./src/middleware.js" must export a function
```

**Fix:** Replace the re-export with an explicit function — see Next.js 16 migration docs.
